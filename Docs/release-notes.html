                                        Release Notes For Codename One - Beta


                                        These release notes are constantly updated for the current beta available to download, they include known issues and pending features.
                                        For up to date details please visit http://www.codenameone.com/

                                        Feature Highlights

                                        Codename One provides several pieces each of which has varying quality statuses and should be considered as such:
                                        • Codename One API - stable beta product
                                        • Codename One Simulator - beta product
                                        • Codename One Designer - beta product. It is recommended you back up your resource files frequently!
                                        • Codename One Plugin - beta product.
                                        • Codename One Cloud/Build Servers - beta product
                                        • iOS Port - Alpha product
                                        • Android port - beta product
                                        • RIM Port - beta product
                                        • J2ME Port - stable beta product
                                        • Windows Phone 7 Port - Unreleased at this time
